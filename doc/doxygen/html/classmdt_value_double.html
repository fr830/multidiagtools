Detailed Description

Contains a floating (double) value with some attributes.

Most of cases, using standard C++ types are sufficent.

When dealing with instruments, some additionnal flags are needed:

  • +OL, -OL: to tell, for example, that a measure is out of instrument's range.
  • Null: for example, if a error occured, a Null value could be returned (from a function).

Note: Null and NaN is not the same. A Null mdtValueDouble contains a NaN value, but a non Null mdtValue can also contain a NaN value. For example, if a communication error occured, a function should return a Null value.
