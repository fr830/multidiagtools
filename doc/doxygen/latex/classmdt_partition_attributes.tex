Get the root path. 

For example, if /usr is on partition / , and /usr was passed to \hyperlink{classmdt_partition_attributes_a2d8e383af725587f0144f573da0f2bd3}{setPath()} , this method will return /\par
 Similar on Wondows, if C: was passed to \hyperlink{classmdt_partition_attributes_a2d8e383af725587f0144f573da0f2bd3}{setPath()} , C:/ is returned.\par
 On Windows, if nativeFormat is true, the $\backslash$ separator is used, else, POSIX / separator is used in returned path.
