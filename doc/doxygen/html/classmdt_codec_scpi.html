mdtValueDouble mdtCodecScpi::decodeValueDouble ( const QByteArray &  data)

Decode a decimal value.

See SCPI 1999.0 , volume 3, section 3.4.1, 3.4.2 3.4.4, 3.4.5 and 3.4.6 Note: this function assumes decimal format, not octal and not hexadecimal.

If data contains a value <= -9.9e37 or >= 9.9e37, returned value will have OL flag set, and also contains a infinity value. If data contains a value == 9.91e37, returned value will be NaN See SCPI 1999.0 , volume 1, section 7.2.1.4 and 7.2.1.5 for details.
