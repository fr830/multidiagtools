Add a column to columns sort order. 

Columns sort order is similar meaning than S\-Q\-L O\-R\-D\-E\-R B\-Y clause.

For example, to sort columns in order 0, 2, 1 all ascending, call\-:
\begin{DoxyItemize}
\item \hyperlink{classmdt_sort_filter_proxy_model_aad45b85d23fe3c2eb6198a9eece685d2}{clear\-Columns\-Sort\-Order()};
\item add\-Column\-To\-Sort\-Order(0, Qt\-::\-Ascending\-Order);
\item add\-Column\-To\-Sort\-Order(2, Qt\-::\-Ascending\-Order);
\item add\-Column\-To\-Sort\-Order(1, Qt\-::\-Ascending\-Order);
\end{DoxyItemize}

Note\-: if given column is out of bound, it will simply be ignored.
