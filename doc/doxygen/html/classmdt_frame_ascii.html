int mdtFrameAscii::putData ( const char *  data,
int  len 
)
virtual

Put data into the frame.

Must be implemented by specific frame subclass.

This method is called by the read thread.

Default implementation simply stores data, and set the EOF condition to true. This way, it's possible to use mdtFrame for raw binary transfers.

Parameters
dataValid pointer to the data
lenMaximum number of byte to store, must not be more than data size
Returns
Number of bytes that could really be stored
Precondition
The data pointer must be valid, and not be the internal pointer returned by data().
