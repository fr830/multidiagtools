QVariant mdtFrameCodecScpiU3606A::decodeSingleValueDouble ( QByteArray &  frame)

Decode a signle value in double floating format.

This method returns a QVariant. QVariant's internal data is converted to double. If value is not valid, the QVariant is cleared, so it is possible to check this with QVariant::isValid(). When decode was done, the values are available with values()
