QStringList mdtPartitionAttributes::availablePartitions ( const QStringList &  ignoreList = QStringList())
static

Get the list of system's available partitions.

On Linux, this return mounted partitions (reading /etc/mtab)
Note: could be fixed in future, when I understand something in DBus :)
On Windows, returns the paths geven by QDir::drives()

Parameters
ignoreListAll paths listed here will not be returned (ignore list can contain / , C:/ ,... , but NOT C:)

QString mdtPartitionAttributes::rootPath ( bool  nativeFormat = false)

Get the root path.

For example, if /usr is on partition / , and /usr was passed to setPath() , this method will return /
Similar on Wondows, if C: was passed to setPath() , C:/ is returned.
On Windows, if nativeFormat is true, the \ separator is used, else, POSIX / separator is used in returned path.
