\subsection{Detailed Description}
Create a temporary table with direct links informations. 

Real links are stored in Link\-\_\-tbl. Link\-List\-\_\-view is also avalable to get more informations.

Link\-\_\-tbl references all links that exists in practice, and also all connections. With \hyperlink{classmdt_cl_path_graph}{mdt\-Cl\-Path\-Graph}, it's possible get the path, including several linked links, starting from a defined unit connection. In some cases, it can be more interesting to have a synthetic view of paths starting from a element. Typical usage could be to see witch connection from a starting connector goes to witch connection of a other connector. This result will also contain several, so called, direct links.
