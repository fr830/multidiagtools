\subsection{Detailed Description}
Representation of a analog input. 

On wide range of automation application, it is usefull to display a state of a input to the user. For example, a analog input of a I/O module, with a 8 bit encoding could be displayed with this class.

This class is used to display a analog input as GUI. \begin{DoxySeeAlso}{See also}
