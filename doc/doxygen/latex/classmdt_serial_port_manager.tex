\subsection{Detailed Description}
Serial port manager class. 

This is the easiest way to use the serial port API.

All needed object are created by constructor. To alter configuration, use \hyperlink{classmdt_serial_port_manager_a4b8ab7b9d53966a1887d9ce8557b8416}{config()}. To access the serial port object, use \hyperlink{classmdt_serial_port_manager_aa96937c2123fe4353b948b848d3e064d}{port()}.

Use \hyperlink{classmdt_port_manager_af1fb103ffafc227337a59c7e82f44fbc}{start()} to begin read/write and stop to end.
