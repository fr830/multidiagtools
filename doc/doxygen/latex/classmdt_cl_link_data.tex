\subsection{Detailed Description}
Data container class for link data. 

Permit to exchange data with Link\-\_\-tbl.

This class was made for data edition. It is also mandatory that fields matches Link\-\_\-tbl. To get data from a view, witch also can miss fields, or have different field name, considere Q\-Sql\-Record.
