\subsection{Detailed Description}
Handle the port info comboBox and the device info comboBox. 

After a port scan was done (f.ex. with \hyperlink{classmdt_port_manager_ad56afb411ab5468005fca04767557ece}{mdtPortManager::scan()} ), it is useful to display the result into one or two comboboxes.

The first combobox will then display a list of available port (for.ex. /dev/ttyS0, /dev/ttyS1, ... for serial port, or attached devices list on USB port, etc...).

The second comboBox is not allways useful. It can be used, f.ex. with USB interfaces. If a specific port manager is implemented, it gives a possibility to list devices attached on a specific RS485 port.
