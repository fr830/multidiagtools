void mdtSqlTableViewController::setTableView ( QTableView *  tv,
QAbstractItemDelegate *  delegate = 0 
)

Set table view to control.

To handle states correctly during edition, a item delegate that emits some signals is needed:

  • void dataEditionBegins() : when data editing beginns
  • void dataEditionDone() : when data editing is finished

If no delegate is given, a default will be created.
