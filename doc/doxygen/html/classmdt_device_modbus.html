Search and connect to physical device.

Will connect to device with hardwareNodeId set with setHardwareNodeId() . This method will beginn to scan hosts in cache file (if exists) . If expcted device was not found, the network is also scanned .

If all devices that must be scanned have a known hardwareNodeId, please consider using connectToDevice(const QList<int>&) , wtch can be faster .
