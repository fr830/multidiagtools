\subsection{Detailed Description}
Data container class for unit connection data. 

Permit to echange data with Unit\-Connection\-\_\-tbl.

This class was made for data edition. It is also mandatory that fields matches Unit\-Connection\-\_\-tbl. To get data from a view, witch also can miss fields, or have different field name, considere Q\-Sql\-Record.
