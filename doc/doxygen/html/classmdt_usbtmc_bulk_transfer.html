enum mdtUsbtmcBulkTransfer::SplitAction_t
strong

USBTMC split action.

Helps to deal with USBTMC split transaction defined in USBTMC 1.0 specification, section 4.2.1.1

This is used by bulk transfer callbacks to know if they have something to do (submit a INITIATE or a CHECK control query).

Enumerator
NoAction 

No action to perform

INITIATE_ABORT 

Begin aborting transfer with INITIATE_ABORT_BULK_[OUT/IN]

CHECK_ABORT_STATUS 

Check status with CHECK_ABORT_BULK_[OUT/IN]_STATUS

INITIATE_CLEAR 

Begin INITIATE_CLEAR

CHECK_CLEAR_STATUS 

Send a CHECK_CLEAR_STATUS request

CLEAR_ENDPOINT_HALT 

Send a CLEAR_FEATURE request with wValue = ENDPOINT_HALT
