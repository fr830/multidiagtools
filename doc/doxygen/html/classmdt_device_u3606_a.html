Search and connect to physical device.

Will scan available ports and open the first port that has device attached maching request.

Parameters:
devInfoRequested device's informations. Note: Vendor ID and/or product ID are automatically set. (in other words, a empty mdtDeviceInfo can be passed). Optionnaly, a serial device can be set (usefull if many U3606A devices are connected)
