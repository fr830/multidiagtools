Restore defualt settings.

Default setting are:

  • separator : comma ( , ) as defined in RFC 4180
  • eol : CRLF (\r\n) as defined in RFC 4180
  • dataProtection : none
  • dataProtectionEscapeChar : none
  • comment : none
