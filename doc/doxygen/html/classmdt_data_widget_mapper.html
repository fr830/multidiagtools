void mdtDataWidgetMapper::addMapping ( QWidget *  widget,
int  section 
)

Adds a mapping between a widget and a section from the model.

This method uses the widget name (objectName), and tries to find the field (in database) that has the same name.
If field is found, some attributes are set (f.ex: read only, mandatory field, ...). At last step, QDataWidgetMapper::addMapping() is called.

Precondition:
A valid model must be set with setModel()
widget must be a valid pointer
