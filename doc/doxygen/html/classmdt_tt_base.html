bool mdtTtBase::removeData ( const QString &  tableName,
const mdtSqlTableSelection &  s,
bool  handleTransaction 
)

Remove data in given selection from given table.

A SQL statement will be built with a filter (WHERE clause). For a given row, each field will be added with a AND clause. Each row will be separated with a OR clause.

For example, if we want to remove data from AB_tbl, and have following data in the selection:

RowField nameData
0A_Id_FK1
0B_Id_FK10
1A_Id_FK2
1B_Id_FK20

the query will be: DELETE FROM AB_tbl WHERE (A_Id_FK=1 AND B_Id_FK=10) OR (A_Id_FK=2 AND B_Id_FK=20)

Parameters
tableNameName of table in witch data must be removed.
sSelection (see detailed description).
handleTransactionIf true, a transaction will be initiated internally.
