QString mdtUicNumber::compact6UicNumber ( )

Get 6 digit compact unformated UIC number.

A 6 or 11 digit UIC number must first be set with setNumber().
Note: if a 11 digits UIC number is set, the 6 last digit (without control key) are returned.
F.ex: if 94 85 7 560 256-0 is set, this method returns 560256.

Returns:
6 digit UIC number without space, without control key, or a empty string on error

bool mdtUicNumber::setNumber ( const QString &  uicNumber)

Set the UIC number.

The UIC number passed as argument is first checked. If ok, it is stored as list of digits. Note: some chars are allowed (space, -), but alpha (a-b, A-B) are not valid.
