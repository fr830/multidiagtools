\subsection{Detailed Description}
Helper class for data validation. 

The S\-Q\-L widget controllers witch are usefull to create database G\-U\-I are event driven. A internal state machine tries to keep G\-U\-I coherent. When, f.\-ex., the user wants to save data, a trigger is activated and state changes (if current state permit it) to a sort of saving state. The drawback of this solution is that there is no way to handle the save steps. This class was made to help on such process.

For example, if you need to do some check before the real submit beginns, you can subclass \hyperlink{classmdt_sql_data_validator}{mdt\-Sql\-Data\-Validator} and reimplement \hyperlink{classmdt_sql_data_validator_af291c18e9c8994085ed4664db3fdcb7c}{check\-Before\-Submit()}. If this method fails, the submit is not done.
