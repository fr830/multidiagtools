\subsection{Detailed Description}
Frame codec for Velleman k8055 board. 

The Velleman k8055 is a USB I/O board:\par
 \href{http://www.velleman.eu/products/view/?id=351346}{\tt http://www.velleman.eu/products/view/?id=351346}\par
 Note: I don't know if the new version, called k8055N has the same format. When looking in demo tool, could say: possibly yes, but unshure.

A very usefull help to build this codec was libk8055 library: \href{http://libk8055.sourceforge.net/}{\tt http://libk8055.sourceforge.net/}\par
 Thank to the authors witch \char`\"{}draw\char`\"{} the frames formats.
