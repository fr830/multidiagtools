Add some rows of data in model.

This is a helper method wich uses QSqlTableModel's insert/setData methods. Notes:

  • submitAll() is done internally when all rows are stored in model.
  • This method will fail if edit strategy is not OnManualSubmit (See QSqlTableModel documentations for details).
Parameters:
rowsA list of rows containing data items (columns). Data will be converted regarding fieldMap.
fieldMapCorrespondence between rows field indexes and model indexes. See mdtFieldMap for details.
