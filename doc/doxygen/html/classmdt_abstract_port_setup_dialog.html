Detailed Description

Base class to build a port setup dialog.

The first step before display a setup dialog is to set the port manager with setPortManager()

virtual bool mdtAbstractPortSetupDialog::applySetup ( ) [protected, pure virtual]

Apply setup.

This method must be implemented in subclass.

When Ok or Apply button was clicked, this method will be called. Case of Ok button, if this method returns false, the setup dialog will not be closed.
