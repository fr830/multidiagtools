QStringList mdtPartitionAttributes::availablePartitions ( const QStringList &  ignoreList = QStringList()) [static]

On Linux, this return mounted partitions (reading /etc/mtab)

QString mdtPartitionAttributes::rootPath ( bool  nativeFormat = false)

Get the root path.

For example, if /usr is on partition / , and /usr was passed to setPath() , this method will return /
Similar on Wondows, if C: was passed to setPath() , C:/ is returned.
On Windows, if nativeFormat is true, the \ separator is used, else, POSIX / separator is used in returned path.
