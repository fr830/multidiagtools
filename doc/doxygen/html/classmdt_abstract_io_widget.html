mdtAbstractIoWidget::mdtAbstractIoWidget ( QWidget *  parent = 0)

Constructor.

Note for subclass: Some widgets will be created in this consructor:

  • lbLabel: a QLabel wich display the short label of I/O and the label of I/O as tool tip
  • pbDetails: a QPushButton that will call a dialog wich displays details of I/O The subclass must place these labels into a layout.
