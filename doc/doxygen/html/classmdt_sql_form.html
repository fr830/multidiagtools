Detailed Description

Base class for high level SQL widgets API.

To create SQL GUI's, some clases are available:

  • mdtSqlFormWidget
  • mdtSqlTableWidget
  • mdtSqlRelation

They help to deal with several common problems, but they need several code (creating objects, configure them, handle memory, ...).

This class was made to have less code for common database GUI .

The mainSqlWidget is based on a mdtSqlFormWidget , and is placed on the upper part of the form . When adding child table with addChildTable() , a instance of mdtSqlTableWidget is created, and placed to a QTabWidget . It is also possible to add a non database related widget, using addChildWidget() . In this case, the given widget is simply added to the internal QTabWidget .
