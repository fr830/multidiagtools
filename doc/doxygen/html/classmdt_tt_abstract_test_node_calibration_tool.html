Do setup.

Will setup database controllers and map widgets contained in testNodeFormWidget. All childs contained in widget that have a name prefixed fld_ will be mapped to corresponding fields in TestNode_view. Note that this for a read only access.

Note: connect signals and slots before calling this method, else UI will be incoherent at initial state.
