Detailed Description

Frame codec for Velleman k8055 board.

The Velleman k8055 is a USB I/O board:
http://www.velleman.eu/products/view/?id=351346
Note: I don't know if the new version, called k8055N has the same format. When looking in demo tool, could say: possibly yes, but unshure.

A very usefull help to build this codec was libk8055 library: http://libk8055.sourceforge.net/
Thank to the authors witch "draw" the frames formats.

bool mdtFrameCodecK8055::decode ( QByteArray &  data)

Decode a incoming frame.

After decoding is done, the values are available with mdtFrameCodec::values() with following order:

  • Indexes from 0 to 4: Input 1 to 5 , type: bool
  • Indexes from 5 to 6: Ain 1 to 2 , type: int
  • Indexes from 7 to 8: Counter value 1 to 2 , type: int
