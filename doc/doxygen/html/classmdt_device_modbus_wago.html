bool mdtDeviceModbusWago::detectIos ( mdtDeviceIos *  ios)

Query device about filed bus + I/Os setup.

The detected I/Os are added to given ios, but not added to internal ios container (use setIos() for this). Goal is to be able to detect setup without alter the internal container, so a check could be made before running really, or something else.
