mdtAbstractPort::error_t mdtUsbPortThread::readUntilShortPacketReceived ( int  maxReadTransfers) [protected]

Read until a short frame is received.

Parameters:
maxReadTransfersMaximum of read transfers before return (If device returns never a short frame, this method will return after maxReadTransfers).
Returns:
NoError or a error of type mdtAbstractPort::error_t. If maxReadTransfers is reached, ReadTimeout is returned.
Precondition:
Port must be set with setPort() before using this method.
