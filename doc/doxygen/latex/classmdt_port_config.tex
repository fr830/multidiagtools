Set the minimal time to wait before try to read. 

Internally, the event system is used for read calls.\par
 For exemple, on Linux, the select() call is used.\par
 It can happen that the underlaying system does not support it, and call returns immediately, causing the reader thread to consume all CPU ressource.\par
 In this situation, the solution is to wait sometime before calling the read function. This wat is done by setting this parameter to a value greater than 0.
