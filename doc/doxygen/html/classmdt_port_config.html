void mdtPortConfig::setReadMinWaitTime ( int  minWaitTime)

Set the minimal time to wait before try to read.

Internally, the event system is used for read calls.
For exemple, on Linux, the select() call is used.
It can happen that the underlaying system does not support it, and call returns immediately, causing the reader thread to consume all CPU ressource.
In this situation, the solution is to wait sometime before calling the read function. This wat is done by setting this parameter to a value greater than 0.

Parameters:
minWaitTimeWait time before read [ms]
