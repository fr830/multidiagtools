QVariant mdtFrameCodecScpi::decodeSingleValueInt ( QByteArray &  frame)

Decode a signle value in integer format.

This method returns a QVariant. QVariant's internal data is converted to int. If value is not valid, the QVariant is cleared, so it is possible to check this with QVariant::isValid(). When decode was done, the values are available with values()
