mdtAbstractPort::error_t mdtDeviceModbus::connectToDevice ( const QList< mdtPortInfo * > &  scanResult,
int  hardwareNodeId,
int  bitsCount,
int  startFrom = 0 
)

Search and connect to physical device.

Will try to connect to device listed in scanResult until hardwareNodeId is found.

Parameters:
scanResultList of mdtPortInfo containing a port name in format host:port.
hardwareNodeIdSee mdtModbusTcpPortManager::getHardwareNodeAddress().
bitsCountSee mdtModbusTcpPortManager::getHardwareNodeAddress().
startFromSee mdtModbusTcpPortManager::getHardwareNodeAddress().
Returns:
A error listed in mdtAbstractPort::error_t (NoError on success).
Precondition:
scanResult must contain valid pointers.
Internal port manager not runnig and port must be closed before calling this method.
