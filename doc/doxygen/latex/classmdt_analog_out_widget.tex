\subsection{Detailed Description}
Representation of a analog output. 

On wide range of automation application, it is usefull to display a output to the user. For example, a analog output of a I/O module, with a 8 bit encoding could be displayed with this class.

This class is used to display a analog output as GUI. \begin{DoxySeeAlso}{See also}
