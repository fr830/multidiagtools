Detailed Description

USBTMC frame.

Used by mdtport API to send en receive frames in USBTMC standard.

References:

  • USBTMC Revision 1.0 (2003)
  • http://www.usb.org/developers/devclass_docs#approved
  • http://www.usb.org/developers/devclass_docs/USBTMC_1_006a.zip

USBTMC MsgID.

Enumerator
MSG_ZERO 

Reserved.

DEV_DEP_MSG_OUT 

USBTMC device dependent command message

DEV_DEP_MSG_IN 

USBTMC response message. Represent the REQUEST_DEV_DEP_MSG_IN if frame is sent from host (OUT)

VENDOR_SPECIFIC_OUT 

USBTMC vendor specific command message

VENDOR_SPECIFIC_IN 

USBTMC vendor specific response message on the Bulk-IN endpoint. Represent the REQUEST_VENDOR_SPECIFIC_IN if frame is sent from host (OUT)
