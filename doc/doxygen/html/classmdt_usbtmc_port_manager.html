Detailed Description

USBTMC port manager.

This is the easiest way to use the USBTMC port API.

All needed object are created by constructor. To alter configuration, use config(). To access the usb port object, use port().

Use start() to begin read/write and stop to end.

bool mdtUsbtmcPortManager::sendReadRequest ( )

Send a read request to device.

USBTMC standard need that a read request is sent to device before we can read any data. If you choose to use the mdtPortManager's newReadenFrame() signal, this method must be explicitly called after writeData(). If you use waitReadenFrame(), this method is called automatically.

Returns:
True on success.
