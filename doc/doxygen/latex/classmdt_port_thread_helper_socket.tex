Set socket. 

Take some parameters in port and port's config and make all needed signal/slot connections.

Parameters that are used by port's config are:
\begin{DoxyItemize}
\item connectTimeout
\item connectMaxTry
\item readTimeout
\item writeTimeout
\end{DoxyItemize}

See \hyperlink{classmdt_port_config}{mdtPortConfig} for details.

Port's mutex is not handled by this method, it should be locked before call.

\begin{DoxyPrecond}{Precondition}
socket must be a valid pointer. 

port must be set with \hyperlink{classmdt_port_thread_helper_a4127c0b28dc63df06b5a36a4be8b3bb7}{mdtPortThreadHelper::setPort()}. 
\end{DoxyPrecond}
