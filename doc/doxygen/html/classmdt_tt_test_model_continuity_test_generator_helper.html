Get related connections.

Will get the DUT connection that is linked to given fromTestConnectionId, then search about linked DUT connections. Finally, a list of test connection is built for each DUT connection found.

Parameters
testCableIdID of test cable (TestCable_tbl) to use.
fromTestConnectionIdTest connection from witch to start.
graphGraph object. Cable list must allready be loaded (see mdtClPathGraph::loadLinkList())
okWill be set to true on success, false else.
