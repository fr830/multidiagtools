\subsection{Detailed Description}
Blinking L\-E\-D. 

This class can be helpfull to get a blinking L\-E\-D in a simple way.\par
 If a lot of blinking L\-E\-D's are needed, it is recommanded to use \hyperlink{classmdt_led}{mdt\-Led} objects and one Q\-Timer, and to connect the Q\-Timer\-::timeout() signal to each \hyperlink{classmdt_led_a9e86ff65f2ec1bfb6131e2b9042609e8}{mdt\-Led\-::toggle\-On\-Off()} slot.
