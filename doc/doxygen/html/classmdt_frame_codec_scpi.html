bool mdtFrameCodecScpi::decodeFunctionParameters ( const QByteArray &  data)

Decode reply of type function/parameter.

Typical query that returns <function> <parameters> form is CONFigure?. See SCPI-99 , chap. 3.

After successfull decode, values() contains:

  • Index 0: Function as string
  • Index >1 (if available): parameters in converted form (double, int, ...). Note: if a parameter is a floating value, it's possible that it is invalid (+/-inf, Nan).

value = checkFloatingValueValidity(value.toDouble());
